# Analisys-service

Estructura inicial del proyecto para análisis de caligrafía.

Estructura generada: capa `core`, `application`, `infrastructure`, `scripts`, `data` y `tests`.

Pasos siguientes:

- Implementar modelos ML en `core/models/siamese_model.py`.
- Completar adaptadores en `infrastructure/adapters`.
- Añadir dependencias a `requirements.txt` y probar `scripts/run_api.py`.
